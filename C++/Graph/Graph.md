# Graphs Data Structure

### Operations 
Let the graph data structure be represented by *G*.
- is_adjacent(G,x,y)
- neighbors(G,x)
- add_vertex(G,x)
- remove_vertex(G,x)
- add_edge(G, x,y)
- remove_edge(G,x,y)
- get_vertex_value(G,x)
- set_vertex_value(G, x,y)
- set_edge_value(G,x,y,v)
- get_edge_value(G,x,y)
